# 05 · Social square (post IG / general)

**Uso**: post cuadrado para IG, general social media, podcast cover si
aplica, listening session announcement.

**Resolución**: 1080×1080 mínimo (4000×4000 si va a Spotify Canvas)
**Aspecto**: 1:1 cuadrado
**Formato**: JPG sRGB

---

## Prompt principal — variante drop announcement

```
square 1:1 social media poster, deep cosmic black background, ÆM
logotype large centered upper-third in pixelated phosphor green,
"HELIOPAUSE / TRANSMISSION 01" in monospace VT323 below, hexagram 24
(復 Fù) symbol pixel art centered, "OUT NOW" or "SIGNAL ACTIVE"
coordinate label in monospace at the bottom, monochromatic phosphor
green #a6d65f on cosmic black #0d1014, retro CRT vintage NASA control
panel aesthetic, pixel-perfect edges no antialiasing, generous padding,
minimal composition
```

## Negative

```
[base anterior] + portrait, vertical, landscape, busy composition,
multiple subjects, character, face, photorealistic, gradient, neon,
synthwave, modern UI elements
```

## Settings

- Resolution: **1024×1024**
- Steps: 4
- CFG: 1.0

## Iteración

1. Generar 6-8 imágenes con prompt principal.
2. Probar V1, V2, V3, V4 según necesidad (announcement / mark / concept / now playing).
3. Para serie de posts coherente, mismo seed entre variaciones.
4. Final: 1080×1080 JPG sRGB.

## Tips

- Para **IG feed** que se vea bien: el cuadrado se renderiza a 1080
  pero la **preview en perfil** es 320×320. Mantener elementos clave
  visibles incluso a tamaño chico.
- IG comprime fuerte. Generar a **4000×4000** y downsize a 1080×1080
  → mejor calidad final que generar a 1080.
